## Quick Font Rules

- **Headlines**: Bold (700-900), large (48-96px), tight letter-spacing for impact
- **Subtitles**: Medium weight (300-500), smaller (18-28px), more letter-spacing for elegance
- **Body**: Regular weight (400), small (12-18px), line-height 1.5-1.8 for readability
- **Accents**: ALL CAPS + letter-spacing 4-10px for labels, badges, CTAs
- **Maximum 2 fonts** per design (headline + body)
- **Google Fonts** are loaded automatically — use `fontFamily="Name, sans-serif"` or `fontFamily="Name, serif"`
