# Color Palettes

Pre-curated color schemes organized by mood. Each file contains multiple palettes with hex codes and usage guidance.

## How to Use

1. Pick the file that matches the design's mood
2. Choose a palette from the file
3. Use the hex values for `bgColor`, `backgroundColor`, `color`, `textStrokeColor`, `shadowColor`, `textOutlineColor`, `fillGradient`, and `bgStyle`
4. Keep accent colors sparse — use on 1-2 elements maximum

## Available Files

| File | Mood | Best For |
|------|------|----------|
| `dark-elegant.md` | Premium, luxury, night, mysterious | Fashion, tech, events, nightlife |
| `light-clean.md` | Minimal, airy, professional, fresh | Corporate, SaaS, editorial, wellness |
| `vibrant-playful.md` | Energetic, fun, youthful, bold | Social media, gaming, entertainment, education |
| `minimal-corporate.md` | Trustworthy, restrained, B2B, serious | Business, finance, consulting, legal |

## Palette Structure

Each palette has:
- **Background**: Page bgColor
- **Text Primary**: Main text color
- **Text Secondary**: Supporting text color
- **Accent**: Primary accent for CTAs, highlights
- **Accent 2**: Secondary accent
- **Surface**: Card/panel background color
- **Border**: Subtle border/divider color

Use `bgStyle` with `linear-gradient` or `radial-gradient` for richer backgrounds using the palette colors.
