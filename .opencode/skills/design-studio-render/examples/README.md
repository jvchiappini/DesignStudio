# Examples — Ready-to-Use Projects

Full `.jsx` projects you can use directly or adapt. Each file is a self-contained `<project>` that opens in Design Studio.

## Available Files

| File | Dimensions | Best For |
|------|-----------|----------|
| `social-media-post.jsx` | 1080×1080 | Instagram feed, Facebook post, square ad |
| `youtube-thumbnail.jsx` | 1280×720 | YouTube video thumbnail, cover image |
| `instagram-story.jsx` | 1080×1920 | Instagram/WhatsApp Status, Reels, Stories |
| `business-card.jsx` | 600×340 | Digital business card, contact card |
| `event-poster.jsx` | 1080×1920 | Event flyer, concert poster, announcement |
| `web-banner.jsx` | 1200×600 | Website hero banner, email header, ad banner |
| `logo-design.jsx` | 500×500 | Brand logo, app icon, avatar |

## How to Use

1. Pick the file that matches the user's request format
2. Read it with the `read` tool
3. Adapt coordinates, colors, text, and images to the user's brand/content
4. Output the final JSX

## Variables to Customize

Each example uses placeholders like `COLOR`, `TITLE`, `URL`. Replace these:
- **Colors**: Match brand palette
- **Text**: Use the user's actual content
- **Images**: Replace `picsum.photos` URLs with real images
- **Fonts**: Match brand typography
