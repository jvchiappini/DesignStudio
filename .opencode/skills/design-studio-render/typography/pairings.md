# Font Pairings

12 curated pairings for different design moods. PICK ONE.

## 1. Modern & Clean
```
Headline: Inter (700)      ← modern sans-serif
Body:     Inter (400)      ← same family, safe pairing
```
Best for: Corporate, tech, minimal.

## 2. Bold & Editorial
```
Headline: Oswald (700)     ← condensed sans, great for titles
Body:     system-ui (400)  ← universal readable font
```
Best for: News, magazines, sports.

## 3. Elegant & Classic
```
Headline: Playfair Display (700)  ← serif with high contrast
Body:     system-ui (300)         ← clean contrast
```
Best for: Luxury, fashion, weddings.

## 4. Playful & Friendly
```
Headline: Poppins (700)    ← geometric rounded sans
Body:     Inter (400)      ← neutral body
```
Best for: Social media, education, lifestyle.

## 5. Cinematic & Dramatic
```
Headline: Oswald (700)     ← tall condensed
Accent:   system-ui (300)  ← thin contrast
```
Best for: Movie posters, events, gaming.

## 6. Modern Serif
```
Headline: DM Serif Display (700)  ← modern serif
Body:     Inter (300)              ← light sans contrast
```
Best for: Portfolios, agencies, premium brands.

## 7. Technical & Clean
```
Headline: Montserrat (700)  ← geometric sans
Body:     system-ui (400)   ← universal
```
Best for: SaaS, developer tools, B2B.

## 8. Vintage & Warm
```
Headline: Bitter (700)      ← slab serif, warm
Body:     system-ui (400)   ← neutral
```
Best for: Cafes, bakeries, crafts, travel.

## 9. High Fashion
```
Headline: Playfair Display (700 italic)  ← elegant italic serif
Body:     Inter (300)                     ← thin sans contrast
```
Best for: Fashion, beauty, lifestyle.

## 10. Modern Gothic
```
Headline: Cormorant Garamond (700)  ← elegant serif
Body:     Inter (300)                ← light sans
```
Best for: Literary, editorial, art.

## 11. Sporty & Energetic
```
Headline: Bebas Neue (700)  ← ultra-condensed uppercase
Body:     system-ui (400)   ← readable contrast
```
Best for: Sports, fitness, events.

## 12. Minimal Luxury
```
Headline: Inter (300)       ← thin weight for elegance
Body:     Inter (400)       ← same family
Letter-spacing: 4-8px       ← wide spacing for premium feel
```
Best for: Watches, jewelry, premium products.
