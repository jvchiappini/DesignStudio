# Font Size Hierarchy Guide

Size scales for different media types. Use as a reference when choosing `fontSize` values.

## Social Media (1080×1080, 1080×1920)

| Role | Size | Weight | Use Case |
|------|------|--------|----------|
| Hero Title | 56-80px | 700-900 | Main headline, visible immediately |
| Subtitle | 22-32px | 300-500 | Secondary message |
| Body | 14-20px | 400 | Description text |
| Caption | 10-14px | 300-400 | Credits, small print |
| CTA / Badge | 14-20px | 600-700 | Button label, tag |
| Label | 10-14px | 400-600 | ALL CAPS labels |

## YouTube Thumbnail (1280×720)

| Role | Size | Weight | Use Case |
|------|------|--------|----------|
| Headline | 60-96px | 800-900 | Main hook, left-aligned |
| Subtext | 20-28px | 400 | Supporting line |
| Duration | 28-36px | 700-900 | Time badge |
| Channel | 14-18px | 500 | Channel name |

## Web Banner (1200×600)

| Role | Size | Weight | Use Case |
|------|------|--------|----------|
| Headline | 40-60px | 600-800 | Value proposition |
| Subtext | 16-22px | 300-400 | Supporting detail |
| CTA | 14-18px | 600-700 | Button text |

## Business Card (600×340)

| Role | Size | Weight | Use Case |
|------|------|--------|----------|
| Name | 28-36px | 600-700 | Person's name |
| Title | 12-16px | 400-500 | Job title, ALL CAPS |
| Details | 10-14px | 400 | Contact info |

## Poster (1080×1920)

| Role | Size | Weight | Use Case |
|------|------|--------|----------|
| Pre-title | 12-18px | 400-600 | "PRESENTS" or similar |
| Headline | 72-120px | 800-900 | Event/concert name |
| Date/Time | 20-28px | 300-500 | When it happens |
| Venue | 16-22px | 300-400 | Where it happens |
| CTA | 18-24px | 600-700 | Action button |
| Footer | 10-14px | 300 | Fine print |

## General Rules

- **Headlines**: Bigger than you think. Social media needs 56px+ minimum.
- **Body**: 16-18px is the most readable.
- **ALL CAPS text**: Increase letter-spacing by 2-8px.
- **Light weight (300) on dark backgrounds**: Looks premium.
- **Multiple text elements**: Stack with 20-60px gaps between them.
- **Never** use fontSize below 10px (unreadable).
- **Never** use fontSize above 120px (crops in most containers).
